# CMIP6_data_to_ISMIP6_grid

Tools to interpolate and extrapolate CMIP6 ocean data to the ISMIP6 stereographic grid in the PROTECT project.

Tested for: MPI-ESM1-2-HR, IPSL-CM6-LR.
